The Project

This project is an animated short film that I put together. There is a lot that goes into every animation. Part of this process included the following: coming up with the initial idea, sketching and storyboarding, creating all the different illustrations that were used in the illustration, and the actual animation itself. The animation follows around a duck and his many attempts at breaking free from his shell. Everything that you see in the animation short was done by me.

This was an extremely fun project to do and I'm very happy with the final result but it also had it's difficulties. Animations are very time consuming and it's very hard to stay organized with all the many assets one needs to complete the process. Organization was key in completing this project. Storyboarding helped a ton with this aspect. It's a guideline on what needs to be done next so that it all flows smoothly. Labeling all the assets properly was also very important so that I could easily find the exact asset I needed for each scene or part of the scene.

Time restraints were also a difficult task. Since this was a school project it also had a due date. Working under pressure isn't always ideal but it is realistic as most jobs have a due date. Staying organized and planning the many different parts was essential in getting this project completed in a timely manner and to produce something that I can be proud of.

The Storyboards

Storyboards are crucial for any animation. Much like wireframes in UX Design, the storyboards give you a sense of direction and how the project is going to be laid out. It allows you to do the animation in a way that is more organized and more effective. Without them the project would be a lot harder to navigate through. The storyboards allow you to figure out ahead of time all the assets that are needed so nothing (or most everything) is not forgotten. You can continue to polish the project the more you dive in to it to make the higher fidelity portion of the animation.
